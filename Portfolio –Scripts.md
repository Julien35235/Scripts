# Portfolio – Scripts Bash & PowerShell
**Julien DESPAGNE**

 Objectif : Administration systèmes • Support IT • Automatisation  
 Environnements : Windows & Linux

---

## Présentation
Ce dépôt regroupe **140+ scripts Bash (.sh) et PowerShell (.ps1)** développés
pour automatiser des tâches réelles d’administration système.

Les scripts couvrent :
- la gestion des utilisateurs
- la sécurité
- le réseau
- les sauvegardes
- la maintenance système
- l’automatisation serveur

---

## Compétences démontrées

### Windows / PowerShell
- Gestion des utilisateurs et profils
- Sauvegarde automatique des comptes
- Scripts réseau (IP, SSH)
- Installation et mise à jour de logiciels
- Administration système via CLI
- Sécurisation et configuration Windows

### Linux / Bash
- Scripts de sauvegarde
- Menus interactifs en ligne de commande
- Mise à jour automatique du système
- Installation des services (GLPI, SSH)
- Administration réseau
- Maintenance serveur
